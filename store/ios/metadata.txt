## Review Notes
This is a therapist practice management app designed for licensed mental health professionals.
All client health data (anamnesis, session notes, evaluations) is stored locally on-device using SQLite.
No client data is transmitted to any server. The only external service is Supabase for email/password authentication.
The app does NOT use HealthKit.
